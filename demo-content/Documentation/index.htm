Install and templates of the theme

The theme has several templates that you may use for your needs. They are:

  • Home page (file templ-home.php) - our nice and shiny home page
  • Blog (two templates: templ-blog-one.php and templ-blog-two.php) - to make a Blog
  • About page (templ-about.php)
  • Contact page (templ-contact.php)
  • Page to showcase services (templ-service.php)
  • Portfolio (three templates: templ-portfolio-one.php, templ-portfolio-two.php, templ-portfolio-three.php) - to showcase projects

And some words about recommended sizes of the images. They are:

  • equal or larger then 1366×768 for sliders for the slider on home page
  • equal or larger then 1366x550 for sections with 100% width
  • on home page: 684x684, 684x342 and 342x342 for images in the grid on the home page
  • 100x20 is recommended size for logo

Home page

Make a home page like in our demo is pretty simple. First, create a static page and choose 'Home page' template for this page. Than choose 'a static page' option for Front Page displays settings and choose your created static page:

You are done!

Among sections for home page are: simple slider, two sections to show "about us" information (content can be added via theme options page), unique grid with 6 items of custom content (content can be added via theme options page), "Our Services" block, block with testimonials, block with brands/logotypes, block with a map and contact information. "About" page

Overview

First of all create a page and choose template 'About Page'. Featured image will be added to the page header in the background. You should also add a title and subtitle (via metabox). Than add a title for "Our story" block and content for the left and the right columns (via metabox).

Also, you can enable "Meet Our Team" block (explained below), "Our Partners" block (explained below) and block with Instagram images (you must activate 'Instagram Feed' plugin (free) to be able to use this block!)

The theme creates a new special user role called 'architector'. All users under this role will be shown in this block on 'About' page. So, first of all choose 'architector' role for some users and add an avatar and fill in their profession on their profiles:

Simple slider on home page and Home Slides CPT

The theme has simple slider built-in. Slides of this slider are posts of Home Slides CPT.

The number of slides which can be shown is unlimited. All published posts of Home Slides CPT will be shown in the slider. To hide a slide just draft related post.

We recommend to use 'Intuitive Custom Post Order' plugin to reoder posts. Posts on the top will be shown first in the slider.

Required and recommended plugins

There is some required and some recommended plugins. All the required plugins are bundled with the theme and only required plugins affect the basic functionality of the theme. Recommended plugins just add some additional functionality, they are not neccessary and some of them are not bundled with the theme. You can buy them or download for free (it depends on the particular plugin). 'Recommended' means that we tested how they work with the theme just to be sure that everything will be ok and we think that they can be useful in some particular situations.

Some of the required plugins are paid and bundled with the theme, like 'Uni User Avatar', so you can use them with ease.

Demo content

You can find export file with demo content in the folder 'Demo content' within item's archive. First of all you have to activate 'Bauhaus' theme ('Bauhaus Child' is a betetr option, actually), then activate plugin 'Uni Custom Post Types and Taxonomies'. Otherwise, not all the information (especially custom post types and custom taxonomies) will be imported!. Than go to 'Tools' -> 'Import' admin menu and choose 'WordPress' from the list. You will be asked to install 'WordPress Importer' plugin. Install it and activate. Now choose 'bauhaus.wordpress.2015-09-11.xml' file from your computer and click 'Upload file and import'. Follow the instructions.

Now go to OptionTree -> Settings and paste the code from 'layout.txt' file and hit Import Layouts

Additionally you have to do a couple of things. First, assign menu in Appearance -> Menus and choose 'Home' page as front page of your website under Settings -> Reading admin menu.

You are done!

The demo content doesn't contain images. It contains only general structure and texts.

FAQ

  • Q: Why links to 'Home slides' or 'Brands' (or other custom post types) have been disappeared from the admin menu?
    A: More likely it is because you have changed the name of the theme's folder. You have to keep it 'bauhaus' all the time! Another reason may be that you have created a child theme but haven't added line "Template: bauhaus" in the style.css of your child theme. Than just add such line and the problem is gone.
  • Q: How to change the favicon?
    A: The theme uses 5 files with favicons for different resolutions/devices. They are: favicon.ico, favicon-16×16.png, favicon-32×32.png, apple-touch-icon-144×144.png, apple-touch-icon-152×152.png So, just change them, but keep the same file names.
  • Q: What is the proper way to customize your theme?
    A: First, you should create a child theme. A sample of such child theme already is in item's archive. You can use it for your needs. Second, make all changes ONLY through your child theme. It is not a good idea to make changes in original theme's files because you won't be able to upgrade it. An additional information about managing a child theme can be found here.
  • Q: What is the proper way to upgrade the theme?
    A: You have to completely replace all the old files. It is event better to just delete an entire folder of your current Bauhaus parent theme, then upload the new folder with the theme. Also, we might make a decision to add some functions to 'functions.php' file of the precreated child theme, so it is always a good idea to look what is new added in that file and compare what you have in your current child theme. Of course, if you are using a child theme.
  • Q: How to translate the theme on my language?
    A: The theme translation files are located in 'languages' folder. Make a copy of en_GB.po, rename it to your lang tags, make a translation and save in to 'languages' folder. Also, you have to generate .mo file with the same name as your .po file. Save this file in to 'languages' folder too. You are done!
  • Q: Is this theme retina ready?
    A: In general, yes! This theme uses icons of fontawesome font and they look nice on screens with any resolutions and/or pixels density. If it comes about thumbnails then you can use this plugin https://wordpress.org/plugins/wp-retina-2x/. Activate it, choose best suitable option for you, regenerate thumbnails (needed only at the first time!) and you are done! And remember to use only big and high quality images on your website!
  • Q: Can I auto update this theme from WP admin?
    A: Yes, you can. All you need to do is to install special plugin "Envato WordPress Toolkit". Here is a detailed instruction how to achieve that.
